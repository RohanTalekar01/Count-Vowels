# Count Vowels in a String using Python

This Python program counts the number of vowels present in a string.

Explanation
The string "education" is stored in variable s
count starts from 0
The loop checks every character in the string
If the character is a vowel (a, e, i, o, u), the count increases by 1
Finally, the total number of vowels is printed
